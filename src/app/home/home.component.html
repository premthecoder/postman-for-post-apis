A Post list is being used here as Post table.
It uses APIs built using Node-Express JS which is deployed on: http://localhost:3000/api/posts
You can perform operations on this table(list) by using GET, POST, PUT, DELETE from the menu bar options.
Please note as this list/table is shared among all the users so the list/table might reflect the changes done by those users as well.
As of now, Post table contains the following data:
Id Content
{{post.id}} {{post.content}}
